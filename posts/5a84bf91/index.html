迁移hexo博客到Mac

2024-08-25
- -
博客-杂谈

Mac环境配置

在Mac安装git和node.js

brew install git
brew install node

安装hexo

npm install hexo g

如果出错，改为下面这条：

npm install hexo-cli -g

初始化hexo目录

mkdir blog
cd blog
hexo init
hexo s

打开 localhost:4000 查看是否成功

使用ssh连接或者token（我这里选择了token）

去对应的github生成personal token，在 Setting -> Developer Setting 下
然后将token复制到_config.yml中的对应位置，打码处为token

配置文件转移

将原电脑的blog文件下的文件转移到Mac的blog文件夹

尝试发布文章

hexo d
